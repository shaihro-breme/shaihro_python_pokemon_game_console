Guía de instalación y configuración (SETUP en Español)Estructura del proyectoTu carpeta principal debería verse así:```pokemon_definitive/│├── pokemon_game.py├── combat_ui.py├── battle_ui.py├── balance_system.py├── pokemon_evolution.py├── pokemon_types.py├── pokedex.py├── save_system.py├── pokefile.pkl           # (Se crea automáticamente al descargar la Pokédex)└── saves/                 # (Se crea automáticamente al guardar una partida)```Requisitos previosAntes de comenzar, asegúrate de tener instalado:Python 3.10 o superiorpip (gestor de paquetes de Python)Librería adicional:```bash  pip install requests-html  ```Crear un entorno virtual```bashpython -m venv .venv```Activar entorno:Windows:```bash  .venv\Scripts\activate  ```Linux / macOS:```bash  source .venv/bin/activate  ```Instalar dependencias```bashpip install requests-html```Archivos principales| Archivo | Descripción ||----------|-------------|| pokemon_game.py | Control principal del juego || combat_ui.py | Menús y modo TEXTER || battle_ui.py | Combate, turnos y capturas || balance_system.py | Daño, experiencia y nivel || pokemon_evolution.py | Evoluciones por nivel || pokemon_types.py | Sistema de tipos || pokedex.py | Descarga/carga los datos Pokémon || save_system.py | Guarda y carga partidas |Ejecutar el juego```bashpython pokemon_game.py```Guardado automáticoLas partidas se guardan en `/saves` con nombres como `shaihro_1.pkl`.ConsejosSi no existe `pokefile.pkl`, el juego descargará los datos automáticamente.Modo TEXTER captura los 151 Pokémon al instante.Puedes editar `balance_system.py` para ajustar el daño o experiencia.Setup and Installation Guide (English Version)Project Structure```pokemon_definitive/│├── pokemon_game.py├── combat_ui.py├── battle_ui.py├── balance_system.py├── pokemon_evolution.py├── pokemon_types.py├── pokedex.py├── save_system.py├── .venv```Activate:Windows:```bash  .venv\Scripts\activate  ```Linux / macOS:```bash  source .venv/bin/activate  ```Install Dependencies```bashpip install requests-html```Main Files| File | Description ||------|--------------|| pokemon_game.py | Main control and entry point || combat_ui.py | Menus and TEXTER mode || battle_ui.py | Battle logic, turns, and capture system ||
